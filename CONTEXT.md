# Mentorna_new - Project Context & Strategy

## 🎯 Business Mission
Mentorna_new is a **Premium Product-Led Mentorship Platform** focused on the "AI Transformation Journey" for children and teenagers (ages 8-18). It bridges the gap between traditional education and the AI-agent-led future (Vibe Coding).

## 💰 Business Model Summary
- **Target:** Upper-middle/high-income parents looking for a competitive edge for their children.
- **Offering:** 8-week journey to build and launch real-world digital products using AI.
- **Pricing:** Premium ($329 launch special) with built-in scarcity (first 30 families).
- **Unique Edge:** Uses "Neubrutalism" design to project bold innovation and leverages "Roadmap-as-a-Service" for regional career pathing (US, Saudi, UAE).

## 🏗️ Technical Stack
- **Frontend:** React 18 (Vite) + Tailwind CSS + shadcn/ui.
- **Backend:** Supabase (Auth, DB, Edge Functions).
- **Data:** TanStack React Query + Zod validation.
- **Analytics:** Deep GA4 integration for funnel tracking.

## 🚀 Growth & Repurposing (The Cowboy Connection)
- **Vibe Coding Proof:** Use the `vibe_coding_apps.md` data to create content showing non-technical success.
- **Educational Stacking:** Connect the "4 Pillars" from Session 8/9 to the Mentorna roadmap.
